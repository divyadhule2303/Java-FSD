## Project Description
A full-stack application where coordinators create events and students can register for them. Features include event creation, student registration/login, viewing events, registering, and coordinator management of registrations.

## Tech Stack
- Backend: Java, Spring Boot, Spring Data JPA, MySQL
- Frontend: React (create-react-app)
- Tools: Postman, Git/GitHub
